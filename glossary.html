Glossary

HTML
HyperText Markup Language sets the structure of a webpage
CSS
Cascading Style sheets gives the HTML its appearance
Domain Name
A domain name is a unique address that can be used to get your website in a web browser
Head
Contains non-displayed information about the document
Hyperlinks
These are what connect the web together, by linking different documents together we get a website
